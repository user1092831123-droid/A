Input Text

Tree Output

How to Use

Format your text with indentation: The indentation level determines the tree structure. Each level of indentation creates a nested branch.

Example 1 - Simple List:

Input: Project Frontend React Vue Backend Node.js Python

Example 2 - File Structure:

Input: src/ components/ Button.js Modal.js utils/ helpers.js index.js public/ index.html

Example 3 - Mixed Content:

Input: Company Engineering Frontend Team Alice Bob Backend Team Charlie Sales John Jane

Tips:

  • Use consistent indentation (spaces or tabs)
  • Each new line becomes a node in the tree
  • Deeper indentation = deeper nesting
  • Works with any text format
